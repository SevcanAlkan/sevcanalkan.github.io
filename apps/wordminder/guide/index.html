Wordminder Guide

User Guide

A simple starter guide for using Wordminder. This page is a first documentation route and can grow into fuller help content later.

Overview

Wordminder helps you collect useful words, organize them into language pairs, and practice them in short daily study sessions.

Getting started

  1. Sign in and create a collection for a source and target language.
  2. Add your own words or start from a predefined starter collection.
  3. Open the word list and begin a short practice session.
  4. Use daily reminders to keep your study rhythm consistent.

Collections

Build multiple collections by language pair and switch between them as your study focus changes. Collection details can help you monitor word count, practice count, and success rate.

Words and practice

Add original text, target text, meaning, extended meaning, difficulty, and proficiency information to each word. Practice runs as flashcard-style multiple-choice review with short summaries at the end of each session.

Reminders and preferences

Wordminder supports a daily reminder flow as well as theme and UI language preferences, so the study experience can stay simple and consistent.
